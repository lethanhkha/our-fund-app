# Our Fund App 💰
Ứng dụng quản lý tài chính cá nhân và cặp đôi được thiết kế chuyên biệt, tối ưu hóa trải nghiệm trên cả thiết bị di động (Mobile-first) và máy tính (Desktop Sidebar).

Đặc biệt, ứng dụng tích hợp Sổ ghi Tips - một tính năng được thiết kế riêng dành cho các chuyên viên ngành dịch vụ (Spa, Salon,...) để theo dõi và đối soát tiền thưởng một cách dễ dàng nhất.

✨ Tính năng nổi bật
📊 Quản lý Thu/Chi: Ghi chép giao dịch chi tiết theo ngày và danh mục.

👛 Quản lý Đa ví: Hỗ trợ nhiều ví (Tiền mặt, Ngân hàng,...), dễ dàng chuyển đổi và đặt ví mặc định.

💅 Sổ Tips Chuyên biệt: Quản lý tiền tips của khách hàng với trạng thái "Đang chờ" và "Đã nhận", tích hợp logic tách bạch ngày đối soát và ngày hiển thị.

🎯 Mục tiêu Tiết kiệm: Tạo các quỹ tiết kiệm, thực hiện góp/rút tiền với logic đối soát dòng tiền chặt chẽ với số dư ví.

📈 Báo cáo Trực quan: Thống kê cơ cấu chi tiêu bằng biểu đồ Pie Chart sinh động.

✨ Trải nghiệm Native: Hiệu ứng chuyển trang và tương tác mượt mà nhờ Framer Motion, thiết kế UI/UX hiện đại, thân thiện.

🛠 Công nghệ sử dụng
Framework: Next.js (App Router)

Styling: Tailwind CSS

Animation: Framer Motion

Database & Backend: Supabase (PostgreSQL)

Charts: Recharts

Deployment: GitHub Pages (Static Export with GitHub Actions)
